# perfomance tracker

Academic Performance Tracker
A personal collection of academic tools designed for tracking my progress as a Data Science & Economics student at Northeastern University.
  Purpose
This is a personal academic tracker built specifically for my needs. It includes calculators and planners to help me stay organized throughout my college journey.
  Tools Included
- GPA Calculator
Calculate semester and cumulative GPA by adding courses with their letter grades and credit hours.
- Final Grade Calculator
Determine what grade I need on a final exam to achieve a desired course grade.
- Course Planner
Track my Data Science & Economics major requirements, mark completed courses, and monitor degree progress.
- Live Site
Visit the site: muadabdi06-lab.github.io/academic-performance
  Note
This project is tailored to my specific academic program at Northeastern University (Data Science & Economics, BS). The course planner is pre-loaded with my major requirements and may not be applicable to other programs.

Built with a clean, Apple-inspired UI featuring:

Smooth animations
